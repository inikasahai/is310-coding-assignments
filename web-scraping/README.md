# Fandom Wiki Scraping Homework

## Chosen Wiki
For this assignment, I chose the Scooby Doo Wiki on Fandom. Scooby Doo was my favorite show as a kid.

## What I scraped
I scraped character names and their links from the Scooby Doo characters category page.

## Why this data is useful
This data could be useful for researchers studying things like fan culture, media franchises, childhood shows, and information about fictional characters. Scooby Doo has been around for a long time with many adaptations, so its fandom wiki shows how fans document characters across different versions of the series.
