# Popularity vs Quality in Novels

## What this is
This project looks at the difference between popularity and how “good” a book is.

I used:
- Goodreads data → ratings (what people think)
- NYT bestseller data → popularity (what sells)

## Research Question
I wanted to see if the most popular books are also the highest rated, or if those are actually different things.

## What I found
- Most Goodreads books are rated pretty high
- Some books are really popular but not the highest rated
- Popularity and quality don’t always match
