# CLI Music HW

This script lets users enter an artist, album and date and saves the data to a CSV file.

# What it does
- Displays example album data using the Rich library
- Lets the user enter multiple album records
- Asks the user to confirm each entry
- Re-prompts if the entry is incorrect
- Saves confirmed entries to a CSV file
- Prints the full file path to the saved file

# Example data shown
- The Weeknd: After Hours
- Taylor Swift: 1989
- SZA: SOS

# Requirements
- Python 3
- Rich library
